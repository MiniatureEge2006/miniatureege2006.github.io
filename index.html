Rise and shine, Mr. Freeman. Rise. And. Shine.

This is a multipurpose Discord bot intended to be entertaining, full of freedom, and easy to use.

Commands

join

Join a voice channel.

leave

Leave the current voice channel.

play/p

Play an audio/song from a given URL. Any URL that yt-dlp supports also works.

repeat/p

Repeat the currently playing audio/song.

queue

Display the current queue.

skip

Skip the currently playing audio/song.

stop

Stop the currently playind audio.

clear

Clear the queue.

pause

Pause the currently playing audio.

resume

Resume the currently paused audio.

nowplaying/np

Display information about the currently playing audio/song.

help/h/commands/c/cmds/pleasehelpme

Get a list of my commands.

ping/pong/latency/whatsmylatency

Get info and latency about G-Man.

caption

Caption media.

code

Execute code.

exif/ffprobe

Use FFprobe to extract exif metadata from media.

ffmpeg

Use FFmpeg as if its a CLI!

tutorial

Make an oldschool video tutorial.

imagemagick/magick

Use ImageMagick as if its a CLI!

yt-dlp/youtube-dl/ytdl/youtubedl/ytdlp/download/dl

Use yt-dlp! (for a list of formats use --listformats)

youtube/yt

Search YouTube.

userinfo/user/member/memberinfo

Displays information about a user. Defaults to the author.

serverinfo/server/guild/guildinfo

Displays information about a server. Defaults to the current server.

channelinfo/channel

Displays information about a text channel. Defaults to the current channel.

voiceinfo/vc/vcinfo/voice

Displays information about a voice channel. Defaults to the current voice channel the author is in.

threadinfo/thread

Displays information about a text channel's thread.

messageinfo/msg/message/msginfo

Displays information about a message.

emojiinfo/emoji

Displays information about an emoji.

stickerinfo/sticker

Displays information about a sticker.

inviteinfo

Displays information about an invite code.

permissions/perms

Displays a user's permissions in a channel. Defaults to the current channel and author.

roleinfo/role

Displays information about a role. Defaults to the everyone role.

baninfo

Displays information about a banned user.

botinfo

Displays information about the bot.

weatherinfo/weather

Displays information about the weather in a location.

colorinfo/color

Displays information about a color. Defaults to a random color.

gradientinfo/gradient

Displays information about a gradient. Defaults to a random gradient.

ai

Use G-AI to chat, ask questions, and generate responses.

resetai

Reset the conversation history of G-AI.

remind/reminder

Set a reminder for yourself.

reminders

View your reminders.

serverreminders

View all reminders for the server.

deletereminder/reminderdelete/delreminder

Delete a reminder, optionally an user's reminders if you have manage server permissions.

clearreminders/remindersclear/clrreminders

Clear all reminders for yourself or user/server if you have manage server permissions.

ruserinfo/ruser

Get information about a Roblox user.

rgameinfo/rgame

Get information about a Roblox game.
